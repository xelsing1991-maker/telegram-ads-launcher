# 🚀 Telegram Ads Launcher

Автоматизация для Telegram Ads: скрипт берет ботов или каналы из файлов, заполняет объявления по координатам экрана, проверяет плохие цели и переносит их в отдельные списки.

![Окно Telegram Ads](docs/telegram-ads-window.png)

## ⚡ Быстрая установка из GitHub

Открой PowerShell и выполни:

```powershell
git clone https://github.com/xelsing1991-maker/telegram-ads-launcher.git
cd telegram-ads-launcher
powershell -ExecutionPolicy Bypass -File .\install.ps1
```

Запуск после установки:

```powershell
.\.venv\Scripts\python.exe launcher.py
```

Скачать без git можно через GitHub: `Code -> Download ZIP`.

## 🧭 Главное меню

После запуска `python launcher.py` доступно только 3 режима:

1. ➕ `Добавлять объявления из боты\каналы`
2. 🔎 `Проверить ботов и удалить плохих в low-bots.txt low-channel.txt`
3. 🎯 `Калибровать координаты проверки ботов`

В режимах `1` и `2` лаунчер спросит источник:

- `1` - брать цели из `data/bots.txt`
- `2` - брать цели из `data/channel.txt`

## 📁 Файлы данных

Все рабочие списки лежат в папке `data`:

- `data/bots.txt` - входной список ботов
- `data/channel.txt` - входной список каналов
- `data/low-bots.txt` - плохие боты, которые не прошли проверку
- `data/low-channel.txt` - плохие каналы, которые не прошли проверку
- `data/used-bots.txt` - уже использованные боты
- `data/used-channel.txt` - уже использованные каналы
- `data/pending-bots.txt` - временный хвост ботов, если целей не хватило на объявление
- `data/pending-channel.txt` - временный хвост каналов, если целей не хватило на объявление

Формат строк:

```text
@example_bot
example_channel
https://t.me/example_bot
```

Пустые строки и строки с `#` игнорируются.

## 🤖 Что делает скрипт

### ➕ Режим 1: добавление объявлений

Скрипт:

- берет цели из `data/bots.txt` или `data/channel.txt`
- исключает цели из `used-*` и `low-*`
- группирует ботов по 3 цели, каналы по 5 целей
- открывает форму Telegram Ads по сохраненным координатам
- заполняет `Ad title`, `Ad text`, URL, CPM, бюджет и цели
- проверяет красную ошибку Telegram Ads про недостаточную аудиторию
- плохую цель переносит в `low-bots.txt` или `low-channel.txt`
- хорошую созданную цель записывает в `used-bots.txt` или `used-channel.txt`
- если целей меньше минимума, переносит остаток в `pending-*`

### 🔎 Режим 2: проверка целей

Скрипт:

- берет выбранный список ботов или каналов
- вставляет цель в поле `Target specific bots`
- смотрит область красной ошибки, сохраненную при калибровке
- если ошибка появилась, удаляет цель из исходного файла
- записывает плохую цель в нужный low-файл
- если ошибки нет, цель остается в исходном файле

### 🎯 Режим 3: калибровка

Скрипт сохраняет координаты в `bot_check_coords.json`. Эти координаты зависят от:

- размера окна браузера
- масштаба Windows
- масштаба страницы
- положения Telegram Ads формы
- длины заголовка и описания

## ⚙️ Настройки

Основные настройки находятся в `telegram_ads_auto.py`:

- `PROMOTE_URL` - ссылка, которую скрипт вставляет в поле URL
- `AD_TITLE_PREFIX` - префикс заголовка для ботов, например `Боты`
- `BUDGET` - бюджет по умолчанию, сейчас `0.7`
- `MIN_CPM` - минимальный CPM, сейчас `0.5`
- `AD_TEXTS` - тексты описаний объявлений

В `telegram_ads_screen_auto.py`:

- `MIN_ACCEPTED_TARGETS = 3` - минимум хороших целей для создания объявления
- `SPEED = 2.0` - базовая скорость автоматизации
- `screen_coords.json` - координаты полного заполнения объявления
- `bot_check_coords.json` - координаты проверки плохих целей

Прямой запуск с параметрами:

```powershell
.\.venv\Scripts\python.exe telegram_ads_screen_auto.py --input data\bots.txt --start-block 1 --start-number 1
```

Полезные параметры:

- `--input data\bots.txt` или `--input data\channel.txt` - какой список брать
- `--start-block 1` - с какого блока начинать
- `--start-number 1` - с какого номера начинать заголовки
- `--limit 3` - обработать только 3 объявления
- `--review` - заполнить форму, но не нажимать финальное создание
- `--cpm 0.5` - CPM в TON
- `--budget 0.7` - бюджет в TON
- `--speed 1` - замедлить автоматизацию
- `--no-strict` - не сверять текст после вставки
- `--leave-terms` - не нажимать чекбокс Terms
- `--test-clipboard` - проверить буфер обмена
- `--test-paste` - проверить вставку текста

Проверка целей напрямую:

```powershell
.\.venv\Scripts\python.exe telegram_bot_checker.py --input data\bots.txt --start-block 1 --limit 10
```

Параметры проверки:

- `--calibrate` - снять координаты проверки
- `--input data\bots.txt` или `--input data\channel.txt`
- `--start-block 1` - с какого блока начинать
- `--limit 10` - сколько целей проверить, `0` значит все
- `--skip-create-click` - использовать, если форма New Ad уже открыта

## 🧩 Формат списков с блоками

Можно просто вставить цели списком, по одной строке. Тогда скрипт сам разобьет их на группы.

Также поддерживается формат блоков:

```text
Блок 1
Цена: 0.7 TON --- 3 бота --- 1 показ в день
@bot_one
@bot_two
@bot_three

Блок 2
Цена: 0.7 TON --- 3 бота --- 2 показа в день
@bot_four
@bot_five
@bot_six
```

Если блоки есть, `--start-block` позволяет начать не с начала файла.

## 🎯 Как правильно калибровать

1. Открой Telegram Ads в браузере.
2. Открой форму создания объявления.
3. Вручную заполни `Ad title`.
4. Потом вручную заполни `Ad text`/описание.
5. Только после этого запускай режим `3`.
6. Наводи мышь на точки, которые просит консоль, и нажимай `Enter`.
7. После калибровки не меняй размер окна и масштаб страницы.

Важно: описание объявления должно быть примерно `135-145` символов. Из-за такой длины поля Telegram Ads сдвигаются в ожидаемое место, и координаты остаются стабильными.

## 🧪 Рекомендуемая отладка

1. Установи зависимости через `install.ps1`.
2. Заполни 5-10 целей в `data/bots.txt`.
3. Запусти `launcher.py`.
4. Выбери режим `3` и откалибруй координаты.
5. Выбери режим `2`, источник `1`, лимит `3`.
6. Проверь, что плохие цели уходят в `data/low-bots.txt`.
7. Выбери режим `1` и создай первое объявление.

Экстренная остановка во время автоматизации: `F4`.

## 🛠️ Частые проблемы

- ❌ `No console input` - скрипт был запущен не в интерактивной консоли.
- ❌ Цели вставляются не туда - заново сделай калибровку и не меняй размер окна.
- ❌ Кириллица вставляется неправильно - запусти `--test-clipboard`.
- ❌ Telegram Ads показывает ошибку, но скрипт ее не видит - заново откалибруй область красной ошибки.
- ❌ Слишком быстро кликает - запускай с `--speed 1`.
